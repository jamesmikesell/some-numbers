🎲How to Play

Welcome to the puzzle where logic meets arithmetic. Every board starts completely filled with numbers, but only some of them belong to the final answer. Your mission is to deduce which cells are required and which are impostors.

One truth, one solution. Each puzzle has exactly one valid solution. Find it by pure deduction—no guessing needed.

🎯 The Goal

  • Each row must add up to the gray number at the left of that row.
  • Each column must add up to the gray number at the top of that column.
  • Each color group must add up to the tiny gray number in that group’s top-left cell.

The correct solution is the only way to satisfy all three rules at once.

✋ Core Gameplay

  1. Start state: The grid shows numbers in every cell, but nothing is selected/circled yet.
  2. Select required cells: Tap or swipe up a cell to select (circle) it — “this number is part of the solution.”
  3. Hide unused cells: Swipe down (or right-click on desktop) to remove a number — “this number is not part of the solution.”
  4. Built-in feedback: If you try to keep a number that doesn’t belong or remove a number that is required, your device will vibrate and the fail counter will increase.

🖥️ / 👆 Controls

  • Mobile / Tablet
    • Tap or swipe up → Select (circle) a number.
    • Swipe down → Remove a number.
  • Desktop
    • Left-click → Select (circle)
    • Right-click → Remove

🧠 Tips & Tricks

  • Easy eliminations: If a row must total 2, you can immediately hide any numbers larger than 2 in that row.
  • Cross-check both ways: A choice must fit the row and the column. If it breaks either total, it’s out.
  • Leverage color groups: Use the group’s tiny top-left total to lock in or eliminate cells inside that color.
  • No guessing: Logical deduction will always get you to the unique solution.

🌟 The Big Picture

Think of it as Sudoku’s cousin: instead of fitting numbers into cells, you’re uncovering the truth about which existing numbers belong. Select what must remain, remove what cannot, and when rows, columns, and color groups all add up perfectly—you win. 🎉
